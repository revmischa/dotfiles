Review the mentioned changes using the **code-reviewer** agent.
If no mentions (PRs etc) are provided, use current branch changes.

Read what changed, then launch the code-reviewer agent to thoroughly analyze the changes for:
- Correctness and potential bugs
- Adherence to project conventions
- Type safety issues
- Security concerns
- Test coverage gaps
- Code quality and maintainability

Provide structured feedback with severity levels (blocking, important, suggestion).

Do not post comments anywhere without confirmation.
